# DeFix - Vanilla JS Code Debugger

A purely Vanilla JavaScript and Node.js application for debugging code instantly with Gemini AI.

## Architecture
- **Frontend:** Vanilla JS, HTML5, Vanilla CSS, TailwindCSS (for utility styling), and CodeMirror 6.
- **Backend:** Node.js, Express, and Google Gemini 2.5 SDK.

## Running Locally
1. Run `npm install` in the root directory.
2. Ensure you have a `.env` file inside the `server/` directory with `GEMINI_API_KEY=your_key`.
3. Run `npm start` to boot the unified Node.js deployment server.
4. Navigate to `http://localhost:3000` to use the application.
